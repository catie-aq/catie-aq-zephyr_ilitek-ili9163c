# ILITEK ILI9163C display Driver

ILITEK ILI9163C display driver for Zephyr OS.

## Hardware requirements
- Zest_Display_LCD
- Any Zest Core board

## Usage
This display driver can be used to display and draw text, images, and shapes in highly readable form.
